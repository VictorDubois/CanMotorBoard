Custom PCB to control motors 

Either DC motors using [LMD18200](https://www.ti.com/lit/ds/symlink/lmd18200.pdf), or brushless [C610/M2006](https://www.robot-maker.com/shop/moteurs-et-actionneurs/553-motoreducteur-brushless-p36-m2006-avec-controleur-c610-553.html)

MCU: [STM32 NUCLEO-G431KB](https://www.st.com/en/evaluation-tools/nucleo-g431kb.html), code is here https://github.com/VictorDubois/Motor-board-STM32-CAN
